什么是公网 IP？

公网 IP 是互联网服务识别你当前网络出口的地址。使用家庭宽带、公司网络、手机流量或 VPN 时，公网 IP 都可能不同。

它不是局域网 IP

例如 192.168.x.x、10.x.x.x 通常是内网地址，只在本地网络中使用。本页面显示的是网站服务器看到的公网地址。

查询方式说明

浏览器会请求公开 IP 查询接口，仅用于读取当前公网 IP 和大致网络信息。若接口不可用，可点击重新查询。

提示：地理位置和运营商信息由公开接口估算，可能不完全准确；公网 IP 可能因网络、运营商 NAT、代理或 VPN 而变化。若当前网络不支持 IPv4 或 IPv6，对应项目会显示不可用。
